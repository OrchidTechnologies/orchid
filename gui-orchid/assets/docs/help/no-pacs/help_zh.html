入门

什么是 Orchid？

Orchid 会将您连接至 VPN 市场，并设有多个 VPN 提供商为您服务。 每次连接网络时，系统会从网络中随机为您分配一个提供商。 无论何时，若不喜欢服务，您皆可关闭 Orchid，然后重新打开，以刷新连接并重新分配提供商。 在连接网络并使用 VPN 后，您需要按字节支付服务费用，通过极微支付付款。

如要向 VPN 提供商支付服务费用，本应用将需要访问至少一个 Orchid 帐户。 Orchid 帐户也必须具有适当的最低存款和余额，才能正常运作。 有关所支持区块链的最低存款和余额的更多信息，可以查看此表格，了解帐户充值的最新要求。 建议您为帐户充值足够的金额，确保有四个彩币和 90% 的帐户效率；有关帐户效率和彩币额的更多信息，请点击这里。

如何打开 Orchid？

一旦您拥有至少一个有资金的帐户时，无论是通过购买应用内积分或通过 dApp 为 Orchid 帐户充值，并且您在线路生成器中将其选中，点击主页界面上的连接打开 Orchid，即可看见您的连接状态。

您将需要在首次打开 Orchid 时授予 VPN 权限。 每次打开 Orchid 时，您会随机连接到网络上的一个提供商。 使用按钮关闭 Orchid 后再打开，可刷新连接，并随机再次连接。

如果 Orchid 在连接过程中卡住了怎么办？

当 Orchid 连接时，应用需要向以太坊区块链查询，以获取有关 VPN 提供商的信息。 有时，网络可能会阻止与以太坊区块链进行连接。 在这种情况下，按钮显示“正在连接”，并且，屏幕上会出现一条信息，显示应用正在查询以太坊以随机获得一个提供商。

要解决这个问题，您需要使以太坊区块链的连接能够正常工作。 请由左向右轻扫，进入设置面板，打开“设置”，选择“链设置”。 您需要找到使 Orchid 运行的以太坊 RPC 终结点；使用“测试”按钮测试默认的 RPC 或您贴到方框内的任何 RPC。 点击这里，阅读更多内容。

如何使用线路生成器？

在主页界面上点击“0 跃点线路”，访问“线路生成器”，然后添加至少一个由 Orchid 帐户支持的跃点，以连接 VPN 提供商。

如需将 Orchid 帐户添加为一个跃点，您需要选择对应的 Orchid ID、出资人地址（此地址将匹配您用于创建帐户的钱包地址）以及帐户资金所在的区块链，然后点击保存。

您可以添加多个跃点以形成一个拥有多个独立 VPN 服务器的“洋葱路由”。 Orchid、WireGuard 和 OpenVPN 协议均支持。

如何操作“管理帐户”？

在“管理帐户”中，您可以查看有关 Orchid ID 及其所有关联帐户的详细信息。 您也可使用 Orchid ID 购买有积分的帐户，并在 Orchid dApp 中使用您自己的加密货币资金创建帐户。

在您当前选择的 Orchid ID 下方，将看到与之关联的所有帐户。 您需要连接一个有资金的帐户，以建立一个线路。

有关购买和创建 Orchid 帐户的更多信息，请参阅以下内容。

什么是流量监控器？

流量监控器是一个网络协议分析器，可让您查看有关手机网络连接的信息。 在主页界面点击右上角的交通灯图标，即可访问。

如何访问多跃点、OpenVPN 和 WireGuard？

要使用 OpenVPN 和/或 WireGuard 跃点，请在主页界面上点击“0 跃点线路”，然后点击“添加新跃点”。 在那里，您可以进行 OpenVPN/WireGuard 配置并将它们添加到您的线路中。

Orchid ID 和帐户

是否可以使用加密货币创建 Orchid 帐户？

是的！ Orchid 生态系统有一个 dApp（去中心化应用程序），用户可以绑定钱包，并使用加密货币资金为 Orchid 帐户充值。

您可通过为本应用中创建的 Orchid ID 充值来创建 Orchid 帐户。 您需要在“管理帐户”界面上点击按钮，将 Orchid ID 从本应用复制到 Orchid dApp，绑定一个钱包，然后为帐户充值，即可创建帐户。

要复制 Orchid ID，请在“管理帐户”界面上点击“Orchid ID”下方显示的号码。

如需更多帮助，请阅读帮助文档。

如何导入现有 Orchid 帐户？

一旦从应用内部或其他地方创建帐户后，即可导出和共享该帐户。 要导入现有帐户，请前往“管理帐户”，点击右上角的齿轮图标，接着选择“导入 ID”。 然后，您可以粘贴 Orchid ID。 在您的 Orchid ID 出现在本应用中后，点击齿轮图标，选择“导入帐户”，粘贴或选择“出资人地址”（对应于您用于创建帐户及在本 dAapp 中访问的钱包地址），然后选择帐户资金所在的区块链。

添加完成后，您会看到与该帐户关联的余额和存款。
